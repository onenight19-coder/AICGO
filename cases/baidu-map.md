# Case 001: Baidu Map

## 百度地图为什么赢

### English

Baidu Map became valuable because it entered a high-frequency, high-intent user scenario: navigation and local search.

The product's core value was not only showing a map. It helped users make decisions in real time: where to go, how to get there, how long it would take, and what was nearby.

Its entry point was strong because users already had location-based needs in daily life. As mobile internet grew, map usage moved from occasional desktop search to constant mobile usage.

Propagation came from infrastructure-level utility. The more local businesses, routes, traffic data, and user behavior accumulated, the more useful the map became.

Timing was critical. Smartphones, mobile GPS, local services, and on-demand platforms all expanded at the same time.

### 中文

百度地图的增长势能来自一个高频、高意图的用户场景：导航与本地搜索。

它的核心价值不只是“看地图”，而是帮助用户实时决策：去哪里、怎么去、需要多久、附近有什么。

它的入口很强，因为位置需求天然存在于日常生活中。随着移动互联网普及，地图从低频的电脑查询，变成了高频的手机工具。

它的传播不是传统意义上的社交传播，而是基础设施型增长。商家、路线、交通数据、用户行为越多，地图越有用。

它的时机也非常关键：智能手机、GPS、本地生活服务、出行平台在同一时期快速成熟。

## DGT Analysis

| Dimension | Judgment |
| --- | --- |
| Value | Real-time decision support for location and mobility |
| Entry | Mobile search, navigation, local services |
| Propagation | Data accumulation and ecosystem integration |
| Timing | Smartphone and local service explosion |

## Dominant Variable

Entry.

入口决定了百度地图能否从工具变成移动互联网基础设施。

## Growth Constraint

The product needed continuous data accuracy, merchant coverage, route quality, and ecosystem partnerships.

## Growth Potential

High.

Baidu Map had strong growth potential because it sat at the intersection of user intent, location data, and mobile behavior.
